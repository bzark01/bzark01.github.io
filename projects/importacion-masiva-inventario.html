// el problema

Cargar inventario y catálogo producto por producto no escala

Ajustar existencias o dar de alta un catálogo grande a mano, registro por registro, es lento y propenso a errores de digitación — más aún cuando hay que llenar también los datos fiscales de cada producto para Costa Rica.

Este módulo permite preparar toda la información en un Excel, revisarla ahí antes de subirla, y dejar que Odoo la procese en lotes, con una barra de progreso en tiempo real y un registro que deja trazabilidad de qué se cargó, quién lo hizo y qué filas fallaron.

Funcionalidades clave

  • Carga masiva desde Excel

    Sube un archivo .xlsx o .xls y deja que el sistema procese cientos o miles de filas de una sola vez, con plantilla de ejemplo descargable.

  • Ajuste de existencias por ubicación

    Actualiza las cantidades a mano de cada producto en la ubicación indicada, con opción de aplicar de una vez o dejar el ajuste listo para revisión.

  • Alta y actualización masiva de productos

    Crea o actualiza fichas completas: nombre, referencia, precios, categoría, impuestos, etiquetas y unidad de medida, todo desde el mismo archivo.

  • Campos fiscales de Costa Rica

    Incluye código CABYS, actividad económica y tipo de código en la misma carga, listos para la facturación electrónica.

  • Registro y auditoría de importaciones

    Cada carga queda guardada con usuario, fecha y archivo, y el detalle fila por fila de cualquier error para corregir y reintentar sin perder el resto.

  • Procesamiento por lotes con barra de progreso

    La información se procesa en bloques configurables, con avance visible en tiempo real y posibilidad de cancelar una carga en curso.

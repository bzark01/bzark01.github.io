Registro de Garantías de Vehículos

Registro de garantías de recubrimientos e instalaciones sobre vehículos, con un formulario web público donde el cliente o el instalador captura la garantía y el vehículo se autocompleta a partir del número VIN. La gestión interna vive dentro de la aplicación de Flota.

// el problema

Registrar garantías a mano es lento y se pierde la evidencia

Cuando un taller aplica un recubrimiento o una instalación sobre un vehículo, la garantía suele quedar en papel o en una hoja suelta: sin número, sin foto del trabajo y sin forma fácil de ubicarla cuando el cliente reclama meses después.

Este módulo pone el registro en manos del propio cliente o del instalador mediante un formulario web público: se escribe el número VIN, el vehículo se autocompleta desde la base oficial NHTSA, se adjunta la foto de la instalación y la garantía queda numerada y disponible al instante para el personal interno, que la gestiona desde Flota.

// qué incluye

Funcionalidades clave

  • Formulario web público

    El cliente o el instalador registra la garantía sin necesidad de usuario en el sistema.

  • Autocompletado por VIN

    Con el número VIN se completan año, marca y modelo desde la base oficial NHTSA.

  • Evidencia de la instalación

    Cada garantía guarda la foto del trabajo, ampliable a pantalla completa.

  • Numeración automática

    Cada garantía recibe un número único correlativo (WAR/2026/000001).

  • Flujo de estados

    Borrador, Confirmada, Activa, Vencida y Cancelada, con control visual por color.

  • Gestión centralizada

    Toda la información de la garantía (propietario, vehículo, producto, instalador) en una sola ficha.

// el problema

Las contraseñas de tus clientes no deberían vivir en hojas de cálculo

Los equipos de soporte gestionan decenas de instancias de Odoo por cliente, cada una con su usuario y contraseña. Cuando esas credenciales se dispersan en hojas de cálculo, chats o notas personales, se vuelven imposibles de auditar y difíciles de rotar.

Este módulo centraliza todas las credenciales dentro de Odoo: cada consulta queda registrada, cada acceso queda trazado y las contraseñas se pueden rotar de forma automática y programada, conectándose por API a cada instancia.

// qué incluye

Funcionalidades clave

  • Bóveda por cliente

    Un repositorio por cliente con todas sus instancias de producción y staging.

  • Consulta auditada

    Cada vez que alguien revela una contraseña queda registrado quién y cuándo.

  • Rotación automática

    Tareas programadas rotan las contraseñas por API, a diario o cada semana.

  • Historial inmutable

    El registro de accesos no se puede borrar: trazabilidad total de las credenciales.

  • Datos de suscripción

    Licencias, workers, almacenamiento y vencimiento junto a la ficha del cliente.

  • Contraseñas seguras

    Genera claves robustas con mayúsculas, números y símbolos automáticamente.

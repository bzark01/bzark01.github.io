// el problema

Perder la base de datos no debería ser una posibilidad

Los respaldos manuales se olvidan, se hacen a destiempo y a menudo quedan en el mismo servidor que la base: si el disco falla, se pierden la copia y el original a la vez.

Este módulo programa las copias de seguridad para que se generen solas cada día y se guarden en el destino que elijas —local, servidor remoto o la nube—, con borrado automático de las copias viejas y aviso por correo si algo falla.

// qué incluye

Funcionalidades clave

  • Respaldos programados

    Una tarea diaria genera las copias solas, sin intervención manual.

  • 8 destinos

    Local, FTP, SFTP, Google Drive, Dropbox, OneDrive, Nextcloud y Amazon S3.

  • Retención automática

    Borra las copias más viejas según los días que definas.

  • Alertas por correo

    Notifica al responsable cuando un respaldo se genera o falla.

  • Zip o Dump

    Copia completa con adjuntos, o solo la base de datos más liviana.

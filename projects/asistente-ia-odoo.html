// el problema

Odoo guarda mucha información, pero acceder a ella requiere saber dónde buscar

Hacer un reporte custom o entender una métrica suele requerir conocimiento técnico: vistas, filtros, exportar a Excel, fórmulas. La gente de negocio termina pidiéndole los datos al área de IT.

Este módulo cierra esa brecha: cualquier usuario puede preguntar en lenguaje natural y Claude responde con datos reales del ERP, gráficos o reportes generados al momento.

// qué incluye

Funcionalidades clave

  • Chat sobre tus datos

    "¿Cuáles fueron las ventas del mes pasado por vendedor?"

  • Dashboards desde lenguaje natural

    Describe el reporte que quieres y se genera solo.

  • Prompt caching

    Reduce el costo de la API hasta 90% en consultas repetidas.

  • Aprobación humana

    Para acciones críticas (modificar, eliminar, pagar) siempre pide confirmación.

  • Métricas y costos

    Visualiza el uso por usuario y el gasto en la API.
